저는 제 일을 좋아합니다.
좋아하니까 즐기게 되고, 더 잘하고 싶어집니다.
그러다보니 블로그 활동을하여
부족한 지식을 채워나가는 것이 재미있고
문제를 해결하기 위해 사람들과 얘기를 나누곤 합니다.
이러한 과정을 통해 웹제작을 완성하였을때,
성취감을 느낄 수 있어 좋습니다.

제가
일을할 땐

  • 1. 웹 접근성을 우선시합니다.
  • 2. 목표를 달성하기 위해 집요하게 파고듭니다.
  • 3. 개인 보다는 팀, 혹은 단체의 목표를 우선시합니다.
  • 4. 퍼블리싱을 좋아하고 잘하고 싶어 노력합니다.

소개합니다.

  • HTML

    웹 표준과 웹 접근성을 준수하려고 노력하며,
    시맨틱 태그를 사용하여 마크업을 작성합니다.

  • CSS

    원하는 스타일로 코드를 구현하며,
    미디어쿼리를 활용하여
    반응형 웹    을 제작합니다.

  • Javascript

    자바스크립트 활용이 가능하며,
    주로 제이쿼리와 병행하여 활용합니다.

  • JQuery

    웹 제작에 필요한 탭 메뉴, 아코디언, 네비게이션 등을 플러그인 없이 제작가능하며,
    스와이프, 풀페이지와 같은
    라이브러리    를 활용할 수 있습니다.

  • PhotoShop

    포토샵의 기본적인 툴을 사용가능하며,
    이미지에 대한 보정을 할 수 있습니다.
